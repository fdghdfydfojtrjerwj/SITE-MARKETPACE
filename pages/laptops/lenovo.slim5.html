Great performance, but the screen colors could be punchier. I use this machine for coding and light photo editing. Surprisingly capable for casual gaming. Bought this mainly for office work, but the integrated graphics actually handle games like Valorant and Minecraft smoothly at 1080p. The cooling system is efficient, and the fans stay quiet unless you really push it. Docked one star because the webcam quality is a bit grainy in low light.

Oliver A.

Good hardware, but ruined by software glitches. The laptop itself looks great and is very fast, but I've had non-stop issues with the Wi-Fi dropping randomly and Windows sleep mode draining the battery overnight. The pre-installed Lenovo software constantly asks for updates. It's frustrating because the hardware has potential, but the user experience has been annoying.

Marta L.

Sleek, professional, and reliable. I love the minimalist design of the Slim 5—no flashy logos or aggressive styling, so it looks perfect in a business meeting. The port selection is great (USB-C charging is a lifesaver), and it wakes up instantly when I open the lid. It’s a reliable workhorse that looks expensive but isn't.

Tom B.

Decent laptop, but the screen brightness is disappointing. Inside, the specs are great for the price, and the laptop runs very cool. My issue is the screen brightness. It’s only rated for 300 nits, which means if you try to work near a bright window or outdoors, it's very hard to see anything. If you only use it in dark rooms, it’s fine, but keep that in mind.

Looking online, it seems like a common issue with some Lenovo models. Customer service has been difficult to deal with regarding the warranty. Very disappointed with the build quality.
