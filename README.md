#### An online gadget store (smartphones, accessories, new arrivals) featuring a realistic checkout/payment simulation.

<img src="https://github.com/fdghdfydfojtrjerwj/SITE-MARKETPACE/blob/main/Screenshot%202026-06-21%20at%2018.56.07.png"></img>

## ✨ Features

- 🏠 Home page with a hero banner and navigation (Shop, Phones, New Arrivals, Accessories)
- 🛒 Shopping cart
- 👤 User account
- 💳 Realistic payment simulation
- 🔢 Adjustable product quantity before purchase
- 📉 Price tracking for products
- 📱 Responsive layout

## 🛠️ Built With
- HTML5 — semantic markup
- CSS3 — styling and responsiveness
- JavaScript (Vanilla) — all site logic, no external libraries
